# FinTech Transaction API

A simple backend system built using FastAPI and PostgreSQL.

## Features
- Create transaction
- Get transactions
- Update transaction
- Delete transaction

## Tech Stack
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy

## Run Locally
pip install -r requirements.txt
uvicorn main:app --reload
